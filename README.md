# Password-Strength-Checker
* Password Strength Checker

This project is developed as Task 03 of my Skill High Cyber Security Internship.

 Project Overview

The Password Strength Checker is a simple Python-based tool that evaluates the strength of a password based on different security criteria. It helps users understand whether their password is weak, moderate, or strong by checking multiple conditions.

-- Features
Checks the minimum password length.
Detects uppercase and lowercase letters.
Verifies the presence of numbers.
Identifies special characters.
Displays the password strength as Weak, Moderate, or Strong.


-- Technologies Used
Python
Regular Expressions (re module)


-- What I Learned
How password validation works.
Using Python's re module for pattern matching.
Checking different password conditions.
Applying conditional statements to evaluate password strength.
Building a simple security-related application.


-- Outcome

This task helped me understand the importance of creating strong passwords and how password validation is implemented in real-world applications.
